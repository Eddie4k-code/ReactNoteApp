# ReactNoteApp
Simple note application with user authentication made with React, node, mongoosedb.



<h1>Users can</h1>
<ul>
  <li>Register and Login</li>
  <li>Create Notes, which are saved to the database</li>
  <li>Delete Notes</li>  
</ul>
